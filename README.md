# How to build 

macOS only until now, Windows support in the near future.

1. Env

```
brew install cmake
brew install boost --c++11

Qt SDK >= 5.9.1
```

2. moex-cli

- CMake (>=3.9)

3. moex-gui

- CMake (>=3.9)
